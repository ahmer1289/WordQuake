<h2>Game Overview</h2>
•	Game Title: WordQuake
•	Genre: Word Game

<h2>Story and Setting</h2>
•	Setting: Multiple Scenarios – Japanese Room(Base)
•	Characters: Tanuki, Neko, Shiba Inu, Chinchilla, Kitsune

<h2>Gameplay Mechanics</h2>
•	Core Gameplay: Players must input words based on the rules of Shiritori.
•	Controls: Keyboard, Voice
•	Objectives: Stopping the opponent from saying a word.
•	Challenges: Words will spawn in causing damage or protection which can stop opponents from inputting words.
